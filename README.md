# Как запустить фронтенд у себя чтоб всё работало

Сначала склонируйте этот репозиторий

## Установка и запуска

### `npm install`

### `npm start`

Откройте в браузере [http://localhost:3000](http://localhost:3000)

## Билд

### `npm run-script build`
